# ZweitRaumWohnung Kinetic Homepage

Statischer Motion-Prototyp für die neue ZweitRaumWohnung-Startseite.

## Lokal ansehen

```bash
python3 -m http.server 8787
```

Dann öffnen: http://127.0.0.1:8787

## Projektstruktur

- `index.html` — Markup
- `styles.css` — Gestaltung und Layout
- `script.js` — Scroll-/Mouse-/Motion-Logik
- `docs/design-brief.md` — verbindliche Designregeln für weitere AI-/Coding-Iterationen

## Arbeitsregel
Vor jeder Änderung `docs/design-brief.md` lesen. Nur den angefragten Bereich ändern, nicht frei redesignen.
